## Executar os comandos:

``` sh
$ npm install -g ionic cordova
```

$ git clone https://github.com/adrianjunior/pokket-app.git
```

$ npm install ng2-charts --save
```
``` sh
$ npm install chart.js --save
```

## Executar aplicação:

``` sh
$ ionic serve
```
